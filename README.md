# Veo3 Text Batch Runner

Tiện ích Chrome giúp chạy prompt hàng loạt cho Veo3 trong Google Labs Flow
và hỗ trợ quét / click tải nhiều video.

Cách cài:
1. Mở `chrome://extensions`
2. Bật **Developer mode**
3. Nhấn **Load unpacked** và chọn thư mục chứa các file này.
4. Vào `https://labs.google/...` (Flow) – extension sẽ tự chèn Batch UI.
